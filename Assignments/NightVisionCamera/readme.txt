
1. Cross Wire Output x,y List starts from 0x20000100 and values are represented by Blue box
2. 0x20000140 address contains the pixel count for encription
3. 0x20000144 is the starting address of the Pixel values array(INPUT Address)
4. The Red colour box values in the memory map image are the pixel value inputs
5. 0x20000170 address initially contains the start of the Decrypted final values(Array of Decrypted value) and the OUTPUT values are indicated by Green box.
